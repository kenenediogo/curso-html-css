Para usar uma fonte de família externa, devemos usar um dos sites correspondentes como por exemplo o site dafont. Clicamos na font desejada e de seguida fizemos o download da mesma font para o computador. Dentro da tag <style> e colamos o código, e escrevemos o caminho na área do URL, o nome deve ser exatamente igual como está no pasta, e não esquecer de colocar a extensão do ficheiro:

        
            @font-face {
                font-family: 'Love';
                src: url('fonts/love larry ttf.ttf') format(truetype);
                font-weight: normal;
                font-style: normal;
            }

De referenciar que este link de fonte externa possui alguns parâmetros importantes, que devemos saber os tipos de fontes que vamos inserir no site:

        
            Tipos de format()
                - opentype (ots)
                - truetype (ttf)
                - embedded-opentype
                - truetype-aat(Apple Advanced Typography)
                - svg

NOTA: O nome que demos no código da font-family, neste caso 'Love', deve ser escrito também na tag da qual queremos que a fonte seja correspondente.
