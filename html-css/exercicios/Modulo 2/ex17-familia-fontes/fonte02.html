Tipos de Alinhamentos

Subtítulo

Neste momento vamos fazer os tipos de alinhamentos que existem em HTML. Os parametros de Alinhamento de texto são:

            
                body {
                    font: normal 16px Arial, Verdana, serif;
                    text-align: left; - Alinhamento a esquerda
                }
                h1 {
                    text-align: center; - Alinhamento ao Centro
                }
                h2 {
                    text-align: right; - Alinhamento a Direita
                }
                p {
                    text-align: justify; - Alinhamento Justificado
                    text-indent: 30px; - Texto indentado, ou ainda dizendo a abertura do parágrafo.
                }
