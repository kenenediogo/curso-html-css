Criando Sites com HTML e CSS

Também podemos usar os selectores especiais "id", que é representada pelo (#), que só é definida numa tag e a "class" que é representada por (.), que pode ser definida em várias tags ao mesmo tempo dentro de uma tag: Por exemplo neste título, temos a classe "id=principal" e podemos incluir a classe "class=destaque", unidas em uma só e ele vai pegar as duas formatações sem problema nenhum. Mas neste caso a formatação da class não está a funcionar porque está definido na principal um background.

Exemplo: 

<h1 id="principal" class="destaque">Criando Sites com HTML e CSS</h1>
