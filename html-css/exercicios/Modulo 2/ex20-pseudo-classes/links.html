Pseudo Classes: ->(:)<-

As pseudo classes são palavras que definem estados de elementos dentro da HTML, e nesta vimos três delas: visited, hover e active:

  • a:visited { color: red; } : faz com que o site visitado tenha a cor que se definiu
  • a:hover { color: black; text-decoration; underline; } : É para realçar o texto quando passamos o cursor por cima. Basta levar o cursor por cima do texto sem clicar, ele muda com a cor que definimos no CSS
  • a:active { color: yellow; } : Esta classe usa-se para definir a cor quando vamos clicar no texto antes de largar o click.

De recordar que a letra "a" que está a frente dos atributos, pertence a tag do link, e pode ser substituida por outra tag.

Pseudo Elementos: ->(::)<-

Os pseudo elementos podem agir nas classes, nos elementos, nos ids, eles mexem no conteúdo periférico do elemento: after

  • a::after { content: ' [link] '; text-decoration: none; font-weight: normal; color: darkgrey; : aqui ele mostra a palavra 'link' no fim de cada link, com toda estilização definida em CSS.
  • a::before { content: ' [link] '; : aqui ele mostra a palavra 'link' no princípio de cada link, com toda estilização definida em CSS.

Recomendação, estudar mais sobre os atributos das Pseudo Classes e dos Pseudo Elementos
