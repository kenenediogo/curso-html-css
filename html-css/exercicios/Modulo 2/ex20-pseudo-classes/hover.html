Exemplo de hover

        div > p {
        display: none; -> escondeu o texto que está dentro desse parágrafo.

        -> este código significa que o parágrafo está dentro da div: é o filho, 
 não vai afectar o parágrafo que está fora da div ao estilizar o código em CSS.
        }

Passe o mouse sobre o texto abaixo. Criou-se uma div que faz esconder o texto e que quando passamos o mouse, ele mostra o texto com as referidas configurações

        div hover > p {
            display: block;
            color: white;
            background-color: red;

        -> este código significa que o parágrafo está dentro da div:hover também é o filho do div. 
 O código acima, vai fazer com que o parágrafo que está oculto apareça 
 com as respectivas configurações feitas no CSS.
        }
